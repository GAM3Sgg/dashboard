## Psychology: Why This Campaign Works

**Zeigarnik Effect:** Users have 3 "open loops" (codes 2, 3, 4 coming) = keeps them engaged  
**Recurring Urgency:** New deadline every week (vs single 30-day deadline) = higher redemption  
**Variable Rewards:** Rotating categories = more habit-forming than fixed offer  
**Reciprocity:** Four gifts feel more generous than one = higher loyalty  
**Commitment:** Users who use Week 1 code are psychologically primed to return Week 2+
