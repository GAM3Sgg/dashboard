## Campaign Overview

**Core Strategy:** Leverage Ramadan's psychology of gifting, community, and spiritual reflection to drive Games+ signups through exclusive Noon discount rewards.

**Key Insight:** Ramadan is the Middle East's most gifting-oriented period. Gamers actively buy gaming accessories for themselves and gifting to friends/family. By pairing Games+ premium access with Noon shopping rewards, we create a win-win that feels like a blessing, not a sale.

## 4. DISCOUNT CODE STRATEGY

### Code Mechanics (4-Week Rolling Structure)

**Code Strategy: Weekly Rotating Codes**

Unlike a single discount, the actual offer is **4 unique codes over 4 weeks**, each tied to different product categories and incentive tiers. This creates sustained engagement + discovery.

**Code Structure:**

| Week | Dates | Code Name | Category | New Users | Repeat Users | Max Discount |
|------|-------|-----------|----------|-----------|--------------|--------------|
| 1 | Feb 18‚Äì24 | `RMDNREADY` | Electronics & Gaming Accessories | 15% off | 10% off | 100 AED / 50 AED |
| 2 | Feb 25‚ÄìMar 3 | `RMDNHOME` | TVs, Laptops, Toys & Games | 20% off | 15% off | 100 AED / 75 AED |
| 3 | Mar 4‚Äì10 | `RMDNSTYLE` | Watches, Fragrances, Eyewear | 20% off | 15% off | 100 AED / 75 AED |
| 4 | Mar 11‚Äì19 | `RMDNFINAL` | Phones, Accessories, Travel Gear | 15% off | 10% off | 100 AED / 50 AED |

**Key Mechanics:**
- Each code **valid only during its week** (not stacking, creates weekly check-ins)
- **New vs. repeat user tiers** (encourages first purchase, then repeat)
- **Category rotation** (Week 1: gaming-focused ‚Üí Week 2: broader home/gaming ‚Üí Week 3: lifestyle ‚Üí Week 4: mobile/travel)
- **Percentage + cap structure** (e.g., "15% off up to 100 AED" = feels more generous than absolute value)

**Why This Works Psychologically:**
1. **Zeigarnik Effect:** Open loop every week ("What's next week's code?") keeps users engaged
2. **Goal-Gradient:** 4-week journey feels achievable; users check each week
3. **Variety Drives Discovery:** Different categories appeal to different occasions/moods in Ramadan
4. **New vs. Repeat Incentive:** Reverse funnel‚Äîfirst discount is high-value (acquisition), repeat discounts lower but still generous (retention)
5. **Scarcity (Weekly):** Each code is only valid 7 days, creating urgency to use it

**Valid On:**
- Exact categories per week (as per Noon partnership)
- NOT stackable across weeks (one code per week)
- NOT combinable with other promotions (exclusive)

**Validity Period:**
- Each code valid **only during its assigned week**
- Post-signup email lists all 4 codes upfront (transparency + planning)
- Users can "save" email and reference it throughout Ramadan

## 8. CONVERSION OPTIMIZATION RECOMMENDATIONS

### Banner Sizing & Placement Strategy

**Desktop Banners:**
- **728x90 (Leaderboard):** Best for above-fold placements on partner sites
  - High visibility but smaller font; test 2-3 word headlines
- **300x250 (Medium Rectangle):** Sidebar or mid-content
  - High standard conversion rate; good for social proof
- **970x250 (Billboard):** Premium above-fold placement
  - Most visibility; best for video or animated content

**Mobile Banners:**
- **320x50 (Mobile Banner):** Sticky top or bottom bar
  - Minimum but persistent presence
  - Test: Sticky bottom > sticky top for games audience
- **300x250 (Medium Rectangle):** Center mobile placement
  - Good for full-screen takeovers on Noon app
- **320x480 (Full-width Mobile):** App interstitial
  - High-impact but high friction; use sparingly

**Placement Psychology:**
- **Above fold:** Higher CTR but higher bounces (test trade-off)
- **Sticky top:** Stays visible; test frequency capping (dismiss after 5 sec auto-minimizes)
- **Sticky bottom:** Less aggressive; ideal for mobile gaming (avoids interrupt during gameplay)
- **Scroll-triggered:** Show after 30-50% page scroll (indicates engagement)

### Color Psychology & Visual Hierarchy

**Ramadan Color Triggers:**
- **Emerald Green:** Islamic tradition, prosperity, community
- **Gold/Warm Bronze:** Spirituality, gifting, luxury (premium positioning)
- **White/Cream:** Purity, peace, Ramadan lanterns
- **Deep Blue:** Trust, stability (secondary, if needed)

**CTA Button Optimization:**
- **Color:** Gold or bright emerald (high contrast against background)
- **Size:** 44-48px height (mobile); 36-40px on desktop
- **Text:** White or cream text on colored button
- **Animation:** Subtle pulse or glow effect (draws attention, doesn't distract)

**Visual Hierarchy Order:**
1. Headline (largest, first attention)

## üìä Psychology Deep-Dive: Why 4-Week Rotating Codes Work Better Than Single Offer

### The Psychological Edge

**1. Zeigarnik Effect (Unfinished Tasks)**
- Traditional model: Single code, one-time offer ‚Üí "done"
- Rotating model: 4 codes, 4 weeks ‚Üí Users have 3 unfinished loops
- **Impact:** Users check email, check app, anticipate next week's code
- **Engagement:** Creates weekly touchpoints without extra marketing effort

**2. Temporal Motivation (Deadline Effect)**
- Single code valid "until end of Ramadan" = abstract deadline (30+ days away)
- Weekly codes valid "7 days only" = concrete, immediate deadline
- **Impact:** Week 1 code feels urgent (use it THIS WEEK). Week 4 code feels urgent (LAST WEEK).
- **Result:** Higher redemption per code despite lower per-code discount

**3. Variety Reduction (Choice Paradox)**
- "How should I use my 75 AED discount?" = paralyzing choice
- "This week: Gaming Accessories (15% off). Next week: Home & Laptops (20% off)" = clear, sequential choices
- **Impact:** Users shop when category aligns with their needs/mood
- **Result:** Higher AOV (targeted shopping) + lower choice paralysis

**4. Habit Formation (Variable Rewards)**
- Fixed reward (single code) = less habit-forming
- Variable rewards (rotating categories) = slot-machine psychology (powerful)
- **Impact:** Users develop "check my Games+ app every Monday" habit
- **Bonus:** This drives Games+ app engagement beyond the discount

**5. Reciprocity Amplification**
- Single discount = "you gave me something"
- Four discounts = "you keep giving me things" (stronger reciprocal obligation)
- **Impact:** Users feel more grateful, more likely to refer, more loyal to brand

**6. FOMO Compounding (Fear of Missing Out)**
- Single FOMO window = 30 days (weak, everyone knows deadline)
- Four FOMO windows = every week feels like last chance (strong, creates recurring urgency)
- **Impact:** Week 3's code triggers "I've missed Week 1 and 2, can't miss Week 3"

### The Business Edge

**Retention Multiplier:**
- Single code: Signup ‚Üí redeem ‚Üí churn
- Four codes: Signup ‚Üí redeem Week 1 ‚Üí return for Week 2 ‚Üí return for Week 3 ‚Üí return for Week 4 ‚Üí possible churn
- **Expected multi-code adoption: 30-50% use 2+ codes, 15-25% use 3+ codes, 5-10% use all 4**

**Spend Multiplier:**
- Single use: One Noon order, one redemption event
- Four codes: Four separate shopping trips (if each appeals to different need)
- **Expected total spend: 600-1000 AED/user vs 200-300 AED single-code campaign**

**Data Advantage:**
- Single code: "User bought when they had 75 AED discount"
- Four codes: "User buys gaming in Week 1, home tech in Week 2, style in Week 3" ‚Üí Rich behavioral data for personalization

### Banner Messaging Implication

The 4-week rotating structure should be **front and center** in all messaging:
- NOT: "Get a Noon discount"
- YES: "Unlock 4 weeks of rotating codes"
- Emphasize FOUR, not one
- Emphasize WEEKLY, not just at signup
